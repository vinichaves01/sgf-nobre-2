MÓDULO USUÁRIOS E ACESSOS — SGF NOBRE

1) Supabase > SQL Editor
Execute o arquivo:
supabase/migrations/20260722_usuarios_acessos.sql

2) Supabase > Project Settings > API
Copie a chave service_role (secreta).
NUNCA coloque essa chave em NEXT_PUBLIC_* nem envie para usuários.

3) Vercel > projeto SGF > Settings > Environment Variables
Crie:
SUPABASE_SERVICE_ROLE_KEY = sua chave service_role
Marque Production, Preview e Development.
Depois faça Redeploy.

4) GitHub
Envie todo o conteúdo desta pasta para a branch main.

5) No SGF
Abra Mais > Usuários e crie as contas.
A senha atual nunca pode ser exibida; o administrador pode apenas redefini-la.
